# Crypto Ascension Arena Bot – Skeleton Version

Telegram-native Web3 Competition Platform

This repository contains the **Skeleton Bot** for Crypto Ascension Arena, a Telegram-based platform demo designed to showcase the interface and basic commands only.

⚠️ Note: This is a demo version with dummy data. No real voting, ranking, or economy features are implemented yet.

## Features:
- /start – Home / Lobby
- /daily – Daily Arena (dummy)
- /voteA & /voteB – Dummy voting responses
- /weekly – Weekly Championship leaderboard (dummy)
- /profile – Dummy XP / Level / Badges
- /info – About / Rules / FAQ
- Inline Buttons for navigation

## Tech Stack:
- Node.js / TypeScript
- Mastra Telegram Library
- Hosting: Render.com (recommended)

## Getting Started:
1. Clone the repository
2. Install dependencies: `npm install`
3. Add TELEGRAM_BOT_TOKEN as environment variable
4. Run the bot: `node index.js` (dummy responses only)
